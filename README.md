A simple and addictive "Snake" game written in **Java** using the **LibGDX** framework. The design is inspired by the iconic retro style of Nokia mobile phones.

## ‚ú® Features
* **Classic Gameplay**: Navigate the snake, collect apples, and grow your tail.
* **Score System**: Earn 10 points for every apple consumed.
* **Game Over Mechanics**: The game stops upon self-collision, featuring a "Press SPACE to Restart" screen.
* **Nokia Physics**: The snake can pass through walls and reappear on the opposite side of the screen.
